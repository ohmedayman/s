# سوشيال فاست - منصة إدارة السوشيال ميديا 🚀

منصة متكاملة لإدارة السوشيال ميديا لمتاجر إلكترونية، مبنية بـ React + Tailwind CSS.

## الميزات الرئيسية

### 📊 لوحة التحكم
- إحصائيات سريعة (المتابعين، التفاعل، الوصول، الرسائل)
- رسوم بيانية للوصول ومعدل التفاعل الأسبوعي
- جدول آخر المنشورات وأداؤها

### 📅 تقويم المحتوى
- عرض شهري للمحتوى المجدول
- إضافة/تعديل/حذف منشورات
- تصنيف المنشورات حسب النوع والحالة
- ألوان تمثل حالة المنشور

### 🎨 مكتبة القوالب
- 6 قوالب جاهزة (عرض، منتج، نصيحة، رأي عميل، خلف الكواليس، عد تنازلي)
- فلترة حسب الفئة
- معاينة مباشرة للقالب
- وصف العناصر القابلة للتعديل

### ✍️ كاتب الكابشنات
- إدخال فكرة المنشور
- توليد 3 خيارات كابشن (رسمي، حماسي، عاطفي)
- اقتراح هاشتاجات مناسبة
- نسخ النص بضغطة واحدة

### 💬 إدارة الرسائل
- ردود جاهزة على الاستفسارات الشائعة
- تصنيف الرسائل (سعر، شحن، شكوى، شكر)
- رسائل حديثة بانتظار الرد
- قوالب قابلة للتعديل

### 📈 تحليل الأداء
- إحصائيات شاملة لكل منصة
- أفضل المنشورات أداءً
- أفضل أوقات النشر
- مقارنة الأداء عبر الفترات

## التصميم

- 🌙 تصميم داكن عصري (Dark Mode)
- 🎨 ألوان البراند: Indigo + Violet + Pink
- ✨ تأثيرات Glass Morphism
- 🔄 انتقالات سلسة (Animations)
- 📱 تصميم متجاوب (Responsive)

## التشغيل

1. افتح ملف `index.html` في المتصفح مباشرة
2. لا يحتاج تثبيت أي برامج إضافية
3. جميع المكتبات محملة عبر CDN

## التقنيات المستخدمة

- **React 18** - مكتبة بناء الواجهات
- **Tailwind CSS** - إطار التصميم
- **Babel Standalone** - ترجمة JSX في المتصفح
